cara membuat element html baru dan memasukkannya ke document html

  • Membuat baru createElement
  • menyimpan dalam variable
  • Memasukkan ke document agar muncul di browser apendChild

contoh syntaxnya

let newElement = document.createElement("h2");
newElement.innerHTML = `Ini adalah hasil dari pembuatan element baru`;
document.body.appendChild(newElement);
